## Platform Configuration ‚öôÔ∏è

### Why Configure Platforms?

The platform configuration system allows you to personalize the application to match your actual investment platforms and preferences.

### How to Configure

1. **Access Configuration**
   - Look for the floating widget (‚öôÔ∏è) in the bottom-right corner
   - Click it to open the configuration modal

2. **Set Platform Names**
   - **Primary Savings Platform**: Your main savings platform (e.g., "PiggyVest", "Cowrywise")
   - **Primary Investment Platform**: Your USD investment platform (e.g., "RiseVest", "Bamboo")

3. **Configure Settings**
   - **Default Currency**: Choose between NGN and USD
   - **Exchange Rate**: Set current NGN/USD rate (or use real-time rates)

4. **Save and Apply**
   - Click "Save Configuration"
   - The page will reload to apply your settings
   - All tools will now use your custom platform names

### Benefits

- **Personalized Experience**: See your actual platform names throughout the app
- **Consistent Branding**: All calculations and reports use your preferred names
- **Better Understanding**: Easier to relate calculations to your real investments

## Investment Calculator üìä

### Overview

The Investment Calculator helps you compare two different investment strategies:

- **Strategy A**: Single-tier compound interest
- **Strategy B**: Two-tier with USD investment

### How to Use

1. **Enter Your Parameters**
   - **Time Period**: How long to invest (1-10 years)
   - **Principal**: Initial investment amount
   - **Monthly Savings**: Regular monthly contributions
   - **Interest Rates**: Set rates for both platforms
   - **Exchange Rate**: Current NGN/USD rate

2. **Choose Rate Options**
   - **Use Real-time Rate**: Automatically fetch current exchange rates
   - **Manual Rate**: Set your own exchange rate

3. **View Results**
   - Compare total returns for both strategies
   - See detailed monthly breakdowns
   - Analyze currency gains from USD appreciation

### Understanding the Results

- **Single-Tier Strategy**: All money stays in your savings platform
- **Two-Tier Strategy**: Interest is transferred to USD investments
- **Currency Gain**: Additional returns from USD appreciation
- **Winner**: Strategy with higher total returns

## 3-Tier Investment Strategy üèóÔ∏è

### Overview

The 3-Tier Strategy combines three different investment approaches:

1. **Savings Platform**: High-yield savings
2. **USD Investment Platform**: Dollar-denominated investments
3. **Vehicle Investments**: Physical asset investments

### How to Use

1. **Configure Each Tier**

   **Savings Platform Tier:**
   - Initial balance
   - Monthly savings amount
   - Interest rate
   - Interest reinvestment percentage

   **USD Investment Platform Tier:**
   - Initial USD balance
   - Interest rate
   - USD appreciation rate
   - Exchange rate

   **Vehicle Investment Tier:**
   - Investment cost per vehicle
   - Return amount per vehicle
   - Investment period
   - Investment cycle (how often to invest)

2. **Set Analysis Period**
   - Choose how many months to analyze (1-120 months)
   - Longer periods show more detailed trends

3. **View Comprehensive Analysis**
   - Overall strategy summary
   - Individual tier performance
   - Monthly breakdown tables
   - Multiple chart visualizations

### Understanding the Strategy

- **Tier 1 (Savings)**: Provides liquidity and steady growth
- **Tier 2 (USD)**: Offers currency diversification and appreciation
- **Tier 3 (Vehicles)**: Generates periodic lump-sum returns

### Key Metrics

- **Total Investment**: Combined amount across all tiers
- **Total Returns**: Overall profit from all strategies
- **USD Portfolio Value**: Dollar-denominated assets
- **Completed Cycles**: Number of vehicle investment cycles

## Troubleshooting üîß

### Common Issues

#### Platform Configuration Not Saving

- **Solution**: Check if localStorage is enabled in your browser
- **Alternative**: Try refreshing the page and reconfiguring

#### Exchange Rates Not Loading

- **Solution**: Check your internet connection
- **Alternative**: Use manual exchange rate entry

#### Calculations Seem Incorrect

- **Check**: Verify all input values are correct
- **Verify**: Ensure currency and rate settings match your scenario
- **Contact**: Report issues with specific details

#### Page Not Loading

- **Solution**: Clear browser cache and cookies
- **Alternative**: Try a different browser
- **Check**: Ensure JavaScript is enabled

### Performance Issues

#### Slow Loading

- **Solution**: Close unnecessary browser tabs
- **Check**: Ensure stable internet connection
- **Alternative**: Try during off-peak hours

#### Charts Not Displaying

- **Solution**: Refresh the page
- **Check**: Ensure browser supports modern JavaScript
- **Alternative**: Try a different browser

### Data Loss Prevention

#### Local Storage Issues

- **Backup**: Export important data regularly
- **Check**: Ensure sufficient browser storage space
- **Alternative**: Use different browsers for testing

#### Configuration Loss

- **Prevention**: Save configurations immediately after changes
- **Recovery**: Reconfigure platforms if needed
- **Backup**: Note down important settings
